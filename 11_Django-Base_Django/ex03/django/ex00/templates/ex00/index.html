Ex00: Markdown Cheatsheet

Headings

# H1
## H2
### H3
#### H4
##### H5
###### H6

Text Formatting

  • Bold: **text** or __text__
  • Italic: *text* or _text_
  • Strikethrough: ~~text~~

Lists

Unordered

- Item 1
- Item 2
  - Subitem

Ordered

1. First
2. Second
3. Third

Links

[Google](https://www.google.com)

Images

![alt text](image.jpg)

Code

Inline

`code`

Block

```python
print("Hello World")
```

Blockquotes

> This is a quote

Horizontal Rule

 ---

Tables

| Name | Age |
|------|-----|
| John | 25 |
| Anna | 30 |

Task Lists

- [x] Done
- [ ] Not done

Escaping Characters

\*Not italic\*
