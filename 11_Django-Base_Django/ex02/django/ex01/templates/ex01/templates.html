Ex01: Template engine.

Django’s template engine provides a powerful way to generate HTML dynamically using a mini-language that bridges the gap between Python and HTML.

1. Displaying Context Variables

Variables passed from the view's context dictionary are rendered using double curly braces. For example: {{ variable_name }}.

2. Blocks

The {% block %} tag defines sections that child templates can override. This is the foundation of template inheritance, allowing us to reuse a base.html structure.

3. Control Structures: Loops

To iterate over a list of data, we use the for ... in loop:

		{% for item in list %}
			<p>{{ item }}</p>
		{% endfor %}

4. Control Structures: If statements

Conditional logic is handled by the if tag, allowing us to display content only when certain conditions are met:

		{% if user.is_authenticated %}
			<p>Welcome back!</p>
		{% else %}
			<p>Please log in.</p>
		{% endif %}
